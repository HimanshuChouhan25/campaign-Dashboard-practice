# Campaign Dashboard

A React.js dashboard for managing advertising campaigns with real-time insights and performance metrics.

## Features

- Campaign overview with filtering (All, Active, Paused, Completed)
- Real-time performance metrics (impressions, clicks, spend, CTR)
- Detailed campaign view with double-click interaction
- Responsive design with modern UI

## Quick Start

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the dashboard.

## Available Scripts

- `npm start` - Run development server
- `npm build` - Build for production
- `npm test` - Run tests

## Tech Stack

- React.js
- JavaScript (ES6+)
- CSS3
- REST API integration
